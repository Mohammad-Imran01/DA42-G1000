
This folder contains a ZIP file "icons.zip" with the default aircraft and userpoint icons that can be customized.

----

The file `little_navmap_alternative_nightstyle.ini` contains an alternative night style by merspieler.
Copy this file to `C:\Users\YOURUSERNAME\AppData\Roaming\ABarthel\little_navmap_nightstyle.ini` to use it.

Suiting `[Profile]` section in `little_navmap_mapstyle.ini` for the alternative night style:

```
[Profile]
BackgroundColor=#000000
BackgroundDarkColor=#111111
ElevationScaleDarkPen=#333333, 1, Solid
ElevationScalePen=#333333, 1, Solid
LabelColor=#000000
LabelDarkColor=#000000
LandColor=#222222
LandDarkColor=#222222
LandOutlineDarkPen=#222222, 1, Solid
LandOutlinePen=#222222, 1, Solid
SafeAltLegLineDarkPen=#ee6666, 3, Solid
SafeAltLegLinePen=#ee6666, 3, Solid
SafeAltLineDarkPen=#ee6666, 4, Solid
SafeAltLinePen=#ee6666, 4, Solid
SkyColor=#0000cc
SkyDarkColor=#0000cc
WaypointLineDarkPen=#dddddd, 1, Solid
WaypointLinePen=#dddddd, 1, Solid
```
